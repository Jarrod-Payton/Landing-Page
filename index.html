Good Ol' Days

Good Ol' Days is an application created with the goal of creating memories between family and friends. This application was developed by myself and 4 classmates during my time at Boise Codeworks . This application runs off of a Vue front end, Node Js backend, and a Mongodb database with Firebase utilization for the storing of images. Because it utilizes Mongodb and Firebase as it's databases it allows us to continue to host the project without needing to pay for a database so this application is still usable to this day.

Vault Keeper

Keeper is a social media application that allows the user to post images with captions as well as store other user's posts in a vault (similar to how an album stores images) that can be either private or public. This was a project I solely developer as my final during my time at Boise Codeworks . This application runs a Vue front end, C Sharp back end, and a MySQL database. Because the database is MySQL this project may not be active past the time frame of my free trial.
